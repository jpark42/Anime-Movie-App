Anime Movie API

This application will provide users the ability to access information about their favorite movies within the Anime genre. Users will also be able to create their own profile where they can create lists of their favorite Anime movies.

Features

  • Return a list of all users
  • Return a list of ALL movies to the user
  • Return data (description, director, date of release, whether it's been featured, and image URL) about a single movie by title to the user
  • Return data about a genre (description) by name/title
  • Return data about director (bio, birth year, death year)
  • Allow new users to register
  • Allow users to update their user info
  • Allow users to add a movie to their list of favorites
  • Allow users to remove a movie from their list of favorites
  • Allow existing users to deregister.

API Endpoints

Buisness Logic URL HTTP Method Query Parameters Request body data format Response body data format
Get all Users /users GET - - A JSON object holding data of all users
Get a User by Username /users/:Username GET - - A JSON object holding data of a specific user
